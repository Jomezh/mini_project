# IoT-Based Food Spoilage Detection using VOC Sensors

This project is an ongoing academic project aimed at detecting food spoilage
using volatile organic compound (VOC) gas sensors.

## System Overview
The system uses a Raspberry Pi Zero 2 W as an edge device to collect data from:

- MQ-series gas sensors (VOC detection)
- Temperature and humidity sensors
- Raspberry Pi camera for visual monitoring

Sensor data will be processed using machine learning models
to identify spoilage patterns.
